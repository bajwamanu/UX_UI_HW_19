WEBSITE REDESIGN

ABOUT THE PROJECT

The Oakland Museum of California (OMCA) brings together collections of art, history, and natural science under one roof to tell the extraordinary stories of California and its people. OMCA's groundbreaking exhibits tell the many stories that comprise California with many voices, often drawing on first-person accounts by people who have shaped California's cultural heritage.

However, when we visited the museum’s website we noticed that it looked dated, had redundant links and buttons, did not provide enough information on the mission and looked cumbersome. This is why we decided to define a User Path and redesign websites main page and a few others in specific to our User persona.

OBJECTIVES

  • To create a better User Experience for visitors when navigating through the website and how it aligns with their intentions
  • To make the website more aligned with the marketing goals
  • To include ‘Accessibility’
  • Tailor the website to the specific buyer journey of our persona

USER REASEARCH & IDEATION

Goal

To redesign user path to make the experience better specifically to improve experience in booking a field trip for a class.
After reviewing organizations, we felt this historic institution could use help with redesign to help improve design which in turn could drive more traffic to the museum.

Participants

Our ideal participants will be teachers who will navigate the path we’re trying to improve.

Methodeology

User tests were conducted with 5 participants. I like, I want and What if analysis was done and Prioritization Matrix was worked on based on their feedback.

This helped us in finding the top 3 key takeaways for website redesign:

  • NAVIGATIONAL EFFICIENCY
  • ACCESSIBILITY
  • COLOR AND USER INTERFACE SIMPLIFICATION

I Like, I Want, I Wish

Prioritization Matrix

Findings

High Impact, Low Complexity

  • Clean User Interface
  • Cohesive Design
  • Accessibility
  • Colors related to the brand
  • Easier nav to book virtual tours
  • Information to be blocked better
  • Attention to detail
  • Information on events to be a part of homepage

High Impact, High Complexity

  • Reviews on Visual tours and other events
  • Calendars with visual cues about what's available
  • Everyone felt included
  • Colors related to the brand
  • Better visual representation included of visual tours and experience

User Persona

Competitive Analysis

Competitive analysis was done with the website of American Museum of Natural History.

  • Both used highlights, images and animations to make their website really engaging.
  • Both paid attention to accessibility.
  • There was reiteration of logo, key features on header, homepage as well as footer.

PROTOTYPING & USER TESTING

FIRST STEP : Based on the new User Flow, paper prototypes were created. After validating the main concept, interactive lo-fi wireframes were created using Figma.


SECOND STEP : User tests were generated on the lo-fi version to make sure that new user flow solves the Users pain points.


THIRD STEP : After 100% success on User Tests, the hi-fi prototypes were created.

STYLE TILE


We decided to keep the original brand colors and logo. However, the whole User Interface will now match the branding which is not the case on their original museumca.org website.


Also, we really liked ‘The Museum of Us’ branding motto of the original website and decided to highlight it in a better way.
